# kotlin-string-caser

Kotlin-canonical string manipulation functions to change the case of strings: Pascal, Camel, Kebab, Train, etc.

# Functions available

```kotlin
    toCamelCase 
    toFirstLowerCase
    toKebabCase
    toPascalCase
    toScreamingKebabCase
    toScreamingSnakeCase
    toSentenceCase
    toSlugCase
    toSnakeCase
    toTitleCase
    toTrainCase
    toUpperCamelCase // (alias for toPascalCase)
```
